# InsightStream-NLP

InsightStream is an Intelligent Multi-Source Insights Engine that serves as a real-time market intelligence dashboard. It processes text data to extract Key Insights and uses an LLM to generate executive summaries and sentiment analysis.

## Project Architecture
- **Frontend**: React + Tailwind CSS (Vite)
- **Backend**: FastAPI + Python
- **NLP**: spaCy (NER Extraction)
- **LLM**: Hugging Face / Mock LLM
